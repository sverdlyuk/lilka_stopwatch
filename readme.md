# Секундомір для Lilka

Проста програма секундоміра для ігрової консолі [Lilka](https://lilka.dev). Написана на Lua як приклад для початківців.

## Можливості

- Запуск і зупинка секундоміра
- Запис кіл з часом кола та загальним часом
- Прокрутка списку кіл кнопками вгору/вниз
- Скидання на нуль

## Керування

| Кнопка | Дія |
|--------|-----|
| **A** | Старт / Стоп |
| **B** | Вихід |
| **D** | Записати коло (під час руху) / Скинути на нуль (коли зупинено) |
| **UP** | Прокрутити кола вгору |
| **DOWN** | Прокрутити кола вниз |

## Встановлення

1. Скопіюй файл `stopwatch.lua` на SD-картку
2. Вибери файл у браузері SD-картки на Lilka
3. Запусти програму

## Структура коду

Програма складається з одного файлу `stopwatch.lua` і побудована на трьох стандартних функціях Lilka:

- `lilka.update(delta)` — логіка програми, обробка кнопок
- `lilka.draw()` — малювання екрану кожен кадр
- допоміжні функції `format_time()` та `draw_button()`
